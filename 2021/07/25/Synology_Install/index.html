群晖安装笔记

2021年07月25日 672 字 大概 2 分钟

群晖真的好用，装个玩玩吧

All in one 家庭服务器折腾起来真的很耐玩.大神的话可以装Linux直接撸命令行

但对于像我这样不想动脑子的,装个群晖先准没错,因为它是真的很好用

安装

本文用的版本是DS918,设备是当年250+淘的蜗牛星际(那时买这玩意回来装win10当pc,感觉又穷酸又超值)

u盘引导

  • 准备一台电脑,一台适合装群晖的设备,一个U盘
  • 用DiskGenius删除u盘所有分区
  • 用芯片无忧查看u盘pid和vid并记录
  • 用DiskImage将img文件也就是引导写入u盘
  • 用DiskGenius找到/esp/grub/grub.cfg
  • 修改其中对应的pid和vid，sn和mac也可以随便改点

硬盘引导

  • 两种引导方式二选一
  • 在pe中用DiskGenius删除硬盘所有分区
  • 用DiskImage将img文件也就是引导写入硬盘
  • 装过群晖的可以改一改sn和mac地址防止发生奇怪的事情
  • 第一次装的可以跳过上一步

装系统

  • 连网线重启，进路由器后台看ip地址
  • 手动安装选择准备好的pat文件
  • 等待安装完成然后下一步下一步

基本设置

  • 进设置关闭自动更新
  • 在存储空间管理员新建存储空间
  • 进设置新建共享文件夹
  • 进我的电脑访问\\192.168.xx.xx
  • 映射好网络驱动器

安装qBittorrent

  • 在控制面板/用户账号中启用admin用户，记得改好密码
  • 在用户账号/高级设置里面启动家目录服务，因为到时候装好了，一系列文件都是默认存在homes目录admin里面
  • 在套件中心手动安装网上找的 qBittorrent 套件a1hm
  • 安装完成后点开或者打开ip:8085/,默认用户名admin密码为adminadmin
  • 到此就能用了

优化

  • 更改下载位置
  • 在控制面版里面新建共享文件夹
  • 切记勾选admin用户的写入读取权限
  • 然后就可以在这个盘里面新建文件夹作为下载目录
  • 如果权限不足可能会出现没速度或者下载失败的情况
  • 替换WebUI
  • 我使用的是CzBiX大佬的项目
  • 下载并解压releases里面的包
  • 将public文件夹放到上面新建的盘符中
  • 把该地址填入qbittorrent的 web ui 设置中
  • 滚轮到下面点设置系统就会自动刷新界面
  • 如果出问题了也不要担心，在网址后面加上/api/v2/app/setPreferences?json=%7B%22alternative_webui_enabled%22:false%7D就能回到原来的UI样式
